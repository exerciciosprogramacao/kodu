🔧 Funcionamento dos Cycles

👉 Todos os Cycles seguem exatamente as mesmas 3 regras de comportamento.

👉 Movimento contínuo: Seguem um caminho pré-definido automaticamente ([always] → [move on path]).

👉 Quando destruídos: Adicionam 1 ponto amarelo ao jogador ([health=0] → [+1 yellow points]).

👉 Ao colidir: Se atingirem o castelo, se autodestroem ([bump castle] → [boom me]).

📑Página 2

Você deve copiar e colar todas as programações da página 1 para a página 2.

O que muda?

  • Os tempos para gerar cada inimigo deve diminuir um pouco, você decide quanto.
  • Faça a Fábrica trocar de cor, para termos um indicador visual que a página mudou.
    •     Quando: [ ] Faz: [color] [red]
  • Depois de 60 segundos a página deve ir para a página 3.

📑Página 3

Você deve copiar e colar todas as programações da página 2 para a página 3.

O que muda?

  • Os tempos para gerar cada inimigo deve diminuir um pouco, você decide quanto.
  • Faça a Fábrica sempre ficar trocando de cor, fazendo ela brilhar a todo momento.
    •     Quando: [timer][.25 seconds] Faz: [color] [random]
  • Depois de 60 segundos você deve ganhar o jogo.

Definições

  • Imóvel: ativado
  • Invulnerável: ativado

👉 Página 1: Inicia com ciclos a cada 3s e 10s, mudando para página 2 após 60s.

👉 Página 2: Acelera produção (2s/5s) e muda cor para verde, vai para página 3 em 60s.

👉 Página 3: Ciclos mais rápidos (2s/5s) com cores aleatórias, vitória após 60s.

👉 Padrão: Cada ciclo faz [open] → [create Cycle X] → [close].

👉 Dificuldade: Tempos diminuem a cada página (3s→2s, 10s→5s).

👉 Estado Inicial (Página 1): O personagem aparece dormindo (mostrando "zzzzzzzzzzz") e acorda ao ser clicado, indo para a Página 2.

👉 Modo Combate (Página 2): Quando vê inimigos próximos, atira (blip). Fica visivelmente bravo (ANGRY) quando atacado. Clicar nele consome 10 pontos amarelos. Vira-se automaticamente para enfrentar inimigos e, com pontos suficientes, avança para a Página 3.

👉 Modo Avançado (Página 3): Agora atira mísseis contra inimigos próximos, mantém o comportamento de enfrentamento e fica permanentemente vermelho, indicando estado de alerta máximo.
